# 股票论坛爬虫 - 东方财富版 🚀

一个自动爬取东方财富股票论坛的工具。**无需安装 Python**，全自动运行，每天自动识别"基本面很强"、"拿住会翻倍"等热门关键词，帮助你发现市场讨论热点。

## 📋 功能特性

- ✅ **自动爬取** - 每天自动运行，无需手动操作
- ✅ **关键词识别** - 自动匹配"基本面很强"、"拿住会翻倍"等热词
- ✅ **频率统计** - 统计各关键词出现次数
- ✅ **结果导出** - 自动保存为 CSV 文件（Excel 可打开）
- ✅ **无需安装** - 完全在 GitHub 服务器运行，你的电脑无需操作
- ✅ **24/7 运行** - GitHub 服务器随时执行爬虫任务

## 🎯 使用说明

### 最简单的方式：什么都不用做！

爬虫已经配置好自动运行了。你只需要：

1. **等待自动运行**（每天北京时间 10:00 AM）
2. **查看结果**：
   - 打开仓库主页
   - 点击 **"Actions"** 标签
   - 看到最新的运行记录
   - 点击进去查看详细结果

### 手动运行爬虫（如果不想等）

1. 打开仓库主页：https://github.com/2723713825-gsh/stock-forum-scraper
2. 点击上面的 **"Actions"** 标签
3. 左边选择 **"Stock Forum Scraper"**
4. 点击 **"Run workflow"** 按钮
5. 选择 **"Run workflow"**
6. 等待 2-5 分钟，爬虫执行完成

### 查看爬虫结果

**方式 1：在 GitHub 网页查看**
1. 进入 **"Actions"** 标签
2. 点击最新的运行记录（绿色 ✓ 表示成功）
3. 向下滚动看 **"Artifacts"** 部分
4. 下载 `stock-forum-results.zip` 文件

**方式 2：查看代码文件**
1. 回到仓库主页
2. 点击 **"Code"** 标签
3. 找到 `output` 文件夹
4. 点击 `results.csv` 文件查看内容

## ⚙️ 自定义配置

### 修改搜索关键词

想要搜索不同的关键词？很简单：

1. 打开仓库中的 **`keywords.json`** 文件
2. 点击右上角的 **编辑按钮**（铅笔图标）
3. 修改关键词列表，例如：

```json
{
  "keywords": [
    "基本面很强",
    "拿住会翻倍",
    "底部启动",
    "潜力股",
    "机构看好"
  ]
}
```

4. 点击 **"Commit changes"** 保存

下次运行爬虫时就会使用新的关键词了。

### 修改运行时间表

想改变爬虫每天运行的时间？

1. 打开 `.github/workflows/scraper.yml` 文件
2. 找到这一行：
```yaml
schedule:
  - cron: '0 2 * * *'  # 每天 UTC 时间 2:00（北京时间 10:00）
```
3. 修改时间后保存

**常用时间设置**：
- `0 0 * * *` → 每天 UTC 0:00（北京时间 8:00 AM）
- `0 2 * * *` → 每天 UTC 2:00（北京时间 10:00 AM）
- `0 8 * * *` → 每天 UTC 8:00（北京时间 4:00 PM）
- `0 14 * * *` → 每天 UTC 14:00（北京时间 10:00 PM）

## ❓ 常见问题

**Q: 爬虫多久运行一次？**
A: 默认每天北京时间 10:00 AM 自动运行一次。

**Q: 结果保存在哪里？**
A: 自动保存在仓库的 `output/results.csv` 文件中，同时在 Actions 的 Artifacts 里可下载。

**Q: 爬虫失败了怎么办？**
A: 
1. 进入 **"Actions"** 标签
2. 点击失败的运行记录（红色 ✗）
3. 查看错误日志了解原因
4. 通常是网络问题，等 1 小时后自动重试

**Q: 能爬其他论坛吗？**
A: 目前支持东方财富。后续可以扩展同花顺、雪球等平台。

**Q: 关键词可以自定义吗？**
A: 可以！编辑 `keywords.json` 文件即可。

**Q: 爬虫会被封 IP 吗？**
A: GitHub 自带的 IP 通常不会被封。如果被封，爬虫会自动等待并重试。
